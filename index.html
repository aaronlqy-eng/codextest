2. DS 常见统计问题

偏实战口径：解释概念、知道什么时候用、能说出风险点。

均值、中位数、众数有什么区别？

均值对极端值敏感；中位数更稳健；众数适合描述最常见类别或离散值。收入、房价等偏态分布通常优先看中位数。

方差和标准差表示什么？

它们衡量数据离散程度。标准差与原始单位一致，更容易解释；方差常用于数学推导和模型目标函数。

std = sqrt(var)
什么是 p-value？

在原假设成立的前提下，观察到当前或更极端结果的概率。p-value 小不代表效果大，只说明数据和原假设不太一致。

置信区间怎么解释？

95% 置信区间不是“参数有 95% 概率在区间里”，而是重复抽样时，这种方法构造出的区间约 95% 会覆盖真实参数。

Type I / Type II Error 是什么？

Type I Error 是错拒原假设，也叫 false positive；Type II Error 是没有拒绝错误的原假设，也叫 false negative。

相关性和因果有什么区别？

相关只说明两个变量一起变化；因果要求排除混杂、反向因果和选择偏差。随机实验通常是识别因果最强的方法。

什么时候用 t-test？

比较两组均值差异时常用 t-test。样本独立、近似正态或样本量足够大时更合适；方差不等可用 Welch's t-test。

什么时候用卡方检验？

常用于两个类别变量是否独立，例如渠道和是否转化是否有关。要求每个格子的期望频数不要太小。

A/B Test 需要注意什么？

先定义 primary metric、最小可检测效果、样本量和实验周期。避免中途频繁看结果、多个指标乱试、用户污染和分流不均。

什么是统计功效 power？

Power 是当真实效果存在时，实验能检测出效果的概率。样本量、效果大小、显著性水平和数据噪声都会影响 power。

power = 1 - beta
偏差和方差的 trade-off 是什么？

高偏差模型过于简单，容易欠拟合；高方差模型过度追随训练数据，容易过拟合。正则化、交叉验证和更多数据可帮助平衡。

混杂变量是什么？

混杂变量同时影响自变量和结果变量，会让关联看起来像因果。常见处理方式包括随机化、分层、匹配、回归控制。

CLT 中心极限定理有什么用？

在样本量足够大时，样本均值的分布趋近正态，即使原始数据不正态。这是很多置信区间和假设检验的基础。

样本量越大一定越好吗？

样本量大能降低随机误差，但无法自动解决偏差、数据泄漏、抽样不代表总体、指标定义错误等系统性问题。

Precision 和 Recall 怎么取舍？

误报成本高时重 precision，例如人工审核资源有限；漏报成本高时重 recall，例如欺诈、疾病筛查、安全风控。

ROC-AUC 和 PR-AUC 有什么区别？

ROC-AUC 看整体排序能力，在类别极不平衡时可能过于乐观；PR-AUC 更关注正类表现，适合稀有事件场景。
